한화 품에 안긴 대우조선, ‘대우’ 떼나…사명 변경 추진

    한화그룹이 인수하는 대우조선해양이 사명 변경을 추진한다. 
    대우조선해양에서 대우 ‘간판’을 떼고 한화조선해양으로 새롭게 시작할 것이란 전망이 유력해 보인다. 
    여기에 한화그룹은 엔진 사업 확대를 위해 STX중공업 인수전에 뛰어드는 등 대우조선 재도약 의지를 드러내고 있다. 
    한화그룹 품에 안긴 대우조선의 실적 개선 속도가 빨라질 것이란 기대감이 나오는 이유다. 
    증권업계 등에선 “대우조선이 올해 1분기 흑자 전환에 성공해 본격적으로 수익을 실현할 것”이란 진단이 많다.
